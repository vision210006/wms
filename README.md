## 📌 프로젝트 목표

```sh
 1) 사용자 측면
  (1) 간단한 가입절차를 통하여 계약하여 이용 할 수 있도록 함.
  (2) 사용 중에는 자신의 물건의 위치를 조회할 수 있도록 함.

 2) 관리자 측면
  (1) 관리되고 있는 물건들의 입고 및 출고, 상태를 조회할 수 있도록 함.
```
