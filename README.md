# wms
창고관리 시스템

## 1. 프로젝트 개요
본 프로젝트는 사용자들이 웹사이트를 통하여 창고에 접근하여 물건을 맡기고, 관리자들이 창고를 효율적으로 관리하도록 돕는 프로그램입니다. 

## 2. 프로젝트 목표
 1) 사용자 측면
  (1) 간단한 가입절차를 통하여 계약할수 있도록 함.
  (2) 자신의 물건의 위치를 조회하고, 관리비용을 조회 할 수 있도록 함.

 2) 관리자 측면
  (1) 관리되고 있는 물건들의 입고 및 출고, 상태를 한번에 조회할 수 있도록 함.
  (2) 관리되고 있는 물건들을 통해 얻어지는 수익을 볼 수 있도록 함.
